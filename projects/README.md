# Deep Learning Projects

This directory contains various deep learning and computer vision projects.

## Projects

### 1. Face & Gesture Recognition System
**Directory:** `face-gesture-recognition/`

A real-time computer vision system that combines face recognition with hand gesture detection. The system can identify individuals and their corresponding gestures simultaneously.

**Key Features:**
- Real-time face recognition using LBPH algorithm
- Hand gesture detection (peace, thumbs up, stop)
- Spatial association between faces and gestures
- Multi-person support
- 15-30 FPS performance

**Technologies:** OpenCV, ONNX Runtime, Python

[View Project](./face-gesture-recognition/)
