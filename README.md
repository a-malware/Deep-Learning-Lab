# Deep Learning Lab

A collection of deep learning and computer vision projects developed during my Deep Learning lab course.

## Projects

### 1. [Face & Gesture Recognition System](./projects/face-gesture-recognition/)
Real-time computer vision system combining face recognition with hand gesture detection. Identifies individuals and their gestures simultaneously using LBPH and ONNX neural networks.

**Tech Stack:** Python, OpenCV, ONNX Runtime, NumPy

### 2. [Zoom Virtual Background](./projects/zoom-virtual-background/)
Virtual background implementation for video conferencing applications.
